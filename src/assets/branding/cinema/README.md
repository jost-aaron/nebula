# Nebula Cinema Brand Assets

The production Cinema mark is based on the top-left Five-Blade Play direction
in `docs/cinema-design/nebula-cinema-stellar-aperture-variants.png`.

- `nebula-cinema-symbol.svg` is the transparent primary symbol.
- `nebula-cinema-symbol-background.svg` is the compact dark badge.
- `nebula-cinema-lockup.svg` is the horizontal wordmark.
- `nebula-cinema-app-icon.svg` is the square launcher icon.
- `nebula-cinema-monochrome.svg` is the one-color reproduction mark.
- `png/` contains raster exports for practical UI and distribution use.

The SVG files are canonical. Regenerate PNGs from them rather than editing the
raster files directly.
